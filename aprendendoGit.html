Aprendendo Git.

Conheça alguns comandos importantes do Git.

  • git init
  • git add
  • git commit
  • git status
  • git show
  • git log
  • git branch
  • git checkout
  • git merge
  • git push
  • git clone
  • git pull

Não se preocupe, vamos explicar cada um deles abaixo

  1. git init - Utilizado para inicializar o git em seu sistema.
  2. git add [nome do arquivo] - Adicione ou atualiza uma "linha do tempo".
  3. git commit - Inicia e comenta uma "linha do tempo".
  4. git status - Informa o status das alterações.
  5. git show - Mostra determinados pontos.
  6. git log - Mostra histórico da linha do tempo.
  7. git branch - Cria novas linhas do tempo.
  8. git checkout - Defini em qual branch será adicionada as alterações.
  9. git merge - Unifica duas linhas do tempo.
  10. git push - Envia o repositório local para a nuvem.
  11. git pull - Recebe repositório da nuvem para o sistema local.
  12. git clone - Clona um projeto
